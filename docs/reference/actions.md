# actions

Action classes, CLI option models, and action discovery.

This is the core module of pyscaf-core. It defines the [`Action`][pyscaf_core.actions.Action] base class that all scaffolder plugins must extend, along with [`CLIOption`][pyscaf_core.actions.CLIOption] and [`ChoiceOption`][pyscaf_core.actions.ChoiceOption] for declaring user-facing options.

See the [Creating Actions](../creating-actions.md) guide for a step-by-step tutorial.
