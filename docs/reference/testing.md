# testing

YAML-driven test framework and pytest plugin for action validation.

This module provides infrastructure for testing scaffolder actions using declarative YAML test files. It includes test discovery, a test runner, and a pytest plugin for filtering tests by action name.

See the [Getting Started](../getting-started.md) guide for how to run tests.
